# SEO Rules & Constraints

## Title Tags
- Maximum 60 characters
- Include primary keyword near the beginning
- Include brand name at the end: "| Free Mind Consultancy"
- Every page must have a unique title

## Meta Descriptions
- Maximum 155 characters
- Include a call-to-action
- Include the primary keyword naturally
- Every page must have a unique meta description

## Headings
- One H1 per page, containing the primary keyword
- Use H2–H4 for content hierarchy
- No skipping heading levels (H1 → H3 without H2)

## Content
- Minimum 800 words for service/landing pages
- Minimum 1200 words for blog posts
- Include primary keyword in the first 100 words
- Keyword density: 1–2% (no stuffing)
- Use related/LSI keywords throughout
- Include internal links (minimum 3 per page)
- Include at least one external link to an authoritative source

## Images
- Every image must have descriptive alt text
- Include keywords in alt text where natural
- Use WebP format when possible
- Maximum file size: 200KB for content images, 500KB for hero images
- Use descriptive filenames (not IMG_001.jpg)

## URLs
- Lowercase, hyphen-separated
- Keep under 75 characters
- Include the primary keyword
- No special characters, underscores, or parameters

## Internal Linking
- Every page should be reachable within 3 clicks from the homepage
- Link related service pages to each other
- Blog posts should link to relevant service pages
- Use descriptive anchor text (not "click here")

## Technical
- Page load time under 3 seconds
- Mobile-friendly (responsive design)
- Valid HTML (no broken tags)
- No broken links (404s)
- XML sitemap kept up to date
- robots.txt properly configured
- Schema markup on all service pages (LocalBusiness, Service)
- Canonical tags on every page

## Prohibited
- No keyword stuffing
- No hidden text or links
- No duplicate content across pages
- No auto-generated low-quality content
- No cloaking or doorway pages
- No link schemes
